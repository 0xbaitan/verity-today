 The Windows 11


Tiya Gupta and Prithviraj Jhunjhunwala

July 18, 2021   |   Technology


On 24th June, Microsoft introduced the world to “The Windows of Tomorrow”, which happens to be undoubtedly the truth. With numerous additions, improvements, and developments, the all-new Windows 11 has too much to offer. From gamers & developers to businesses, Windows 11 has introduced the world to something which is the definition of user friendly in today’s world.

For starters, the most noticeable change is the all-new taskbar. The Start Menu has been moved to the centre, allowing users to find what they need faster and in an easier way.



Start now utilises the power of Microsoft 365 and the cloud to show the most recently accessed files, no matter on what platform or device on which we had viewed them earlier, for example, Android or iOS devices. Microsoft has also integrated the taskbar with Chat from Teams which allows you to connect to others instantly.

This new Chat feature allows users to connect through text, chat, voice or video with all their contacts, anywhere, no matter the platform or device the receiver is on. It enables instantaneous communication that has become essential in this gruelling pandemic.

It also allows users to connect with people who don’t have the Teams application downloaded on their devices. It uses a two-way SMS system that gives everyone a way to link devices across Windows, Android or IOS.

Windows 11 introduces to the world an all-new perspective to multitasking. Windows 11 has snap layouts that allow users to divide their screen into multiple sections and place separate windows in each of them. It is a diverse and different version of what we have seen as a split-screen in our devices to date. Users can now divide their screen in new ways making multitasking easier, faster and more organised.

Now, this also brings us to Snap groups which allow users to work on multiple projects at a go. With the ability to save snap layouts on the taskbar, the user can continue working from where they left off on all the specific snap layouts as and when they want. Windows 11 also enables users to create multiple desktops and toggle in between them with ease. In turn, this increases user efficiency, allows easier access, and keeps everything more organised.

Next, we have the new Microsoft Store that now provides more opportunities to creators and developers. Microsoft achieved this by allowing users to introduce their app onto the store, regardless of the app framework and environments they had built them on, such as Win32 or Progressive Web App (PWA).

It is designed for any device and has applications from all brands now. This new store has also brought updated revenue-sharing policies, as now the developers are allowed to keep 100% of the profits, and Microsoft takes 0% away from them.

By virtue, this benefits the users as it gives them secure, frictionless access to the apps, movies, shows, games and web content they need or want. Users are also allowed to download Android apps from the Microsoft app store, for example, TikTok on our laptops.

The devices which will have Windows 11 also unleash a captivating gaming experience for all gamers. The OS now comprises Auto HDR, Xbox Game Pass and better efficiency or processing that helps the game run smoothly. The Auto HDR provides superior graphics. Meanwhile, the Xbox Game Pass provides gamers with access to a vast pool of games through the in-built Xbox app on Windows 11 devices.

Better processing and unparalleled speed achieved by DirectX 12 Ultimate enables lightning-fast processing & direct storage. Thus, improving the gaming experience ten-folds. The concept it uses is that direct storage allows the game to quickly load the assets to the graphics card without slowing down the CPU. It doesn’t just end here! Windows 11 also unlocks the full potential of the user’s system’s hardware and enables breathtaking, immersive graphics at high frame rates.

Another new aspect seen in Windows 11 is the Widgets feature, which appears on the user’s screen like a sheet of glass overlaying the screen display. It was designed in a way to prevent disruption or interruption of the user’s work. It allows a personalised feed and gives a faster way to search and check the ongoings of the world (i.e., latest events, time, weather). The user is no longer required to use their phones to check such things as this Widgets menu allows it all with such ease.

In conclusion, the event held on 24th June 2021 brought to the world a novel innovation that has changed Windows forever. It has become faster, more user-friendly, and more secure than it has ever been. The Windows 11 uses less energy, enhancing battery life, and is also 40% smaller. More efficient updates will become the norm, improving the durability of the device. Windows 11 has brought beauty to technology with new themes available in dark and light modes, texture adjustments and many more design improvements. Its improved voice typing features with automatic punctuation and voice commands brings to humanity technology as never experienced before.
